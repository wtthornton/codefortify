Prerequisites

System Requirements

  • Node.js: Version 18.0.0 or higher
  • npm: Version 8.0.0 or higher (or yarn)
  • Git: For version control integration
  • Operating System: Windows, macOS, or Linux
  • Memory: At least 4GB RAM recommended
  • Storage: 500MB free space for installation

Development Environment

  • Code Editor: VS Code, WebStorm, or any modern editor
  • Terminal: Command line access
  • Browser: Modern browser for documentation and demos

Your First Project

Ready to create your first CodeFortify project? Follow these steps:

Step 1: Initialize Project

Create a new project or use an existing one:

# For new projects
codefortify init my-awesome-project

# For existing projects
cd my-existing-project
codefortify init

Step 2: Analyze Current State

Get your baseline quality score:

codefortify score --detailed

This will show you where your code stands and what can be improved.

Step 3: Start Enhancement

Begin the continuous enhancement process:

codefortify enhance --watch

Watch as AI agents analyze and improve your code automatically!

Step 4: Monitor Progress

Track improvements over time:

codefortify score --compare

See how your code quality improves with each enhancement cycle.

Advanced Commands
