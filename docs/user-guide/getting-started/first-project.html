Option 1: Template Setup (Recommended)

Using a template gives you a solid foundation with best practices already implemented:

Step 1: Initialize from Template

Create a new project using the React TypeScript template:

codefortify template init my-todo-app --template react-typescript

This creates a project with modern React, TypeScript, and Tailwind CSS already configured.

Step 2: Navigate to Project

Move into your new project directory:

cd my-todo-app

You'll see the project structure with all the necessary files and configurations.

Step 3: Install Dependencies

Install all the required dependencies:

npm install

This installs React, TypeScript, Tailwind CSS, and all other dependencies.

Step 4: Get Baseline Score

Analyze the initial code quality:

codefortify score --detailed

This shows you the starting quality score and what can be improved.

Step 5: Start Enhancement

Begin the AI enhancement process:

codefortify enhance --watch

Watch as AI agents analyze and improve your code automatically!

Option 2: From Scratch Setup

Starting from scratch lets you see how CodeFortify guides you through the entire process:

Step 1: Create Project Directory

Create a new directory for your project:

mkdir my-todo-app
cd my-todo-app

This creates a clean slate for your project.

Step 2: Initialize CodeFortify

Initialize CodeFortify in your project:

codefortify init

This creates the CodeFortify configuration and analyzes your project structure.

Step 3: Let AI Guide You

CodeFortify will analyze your project and provide recommendations:

codefortify analyze --suggest

The AI will suggest the best technology stack and project structure for your needs.

Step 4: Apply Recommendations

Review and apply the AI suggestions:

codefortify enhance --apply-suggestions

CodeFortify will automatically set up the recommended structure and dependencies.

Step 5: Start Development

Begin coding with AI assistance:

codefortify enhance --watch

As you code, AI agents will continuously suggest improvements and optimizations.

Option 3: Existing Project Setup

Add CodeFortify to an existing project to enhance what you already have:

Step 1: Navigate to Project

Move to your existing project directory:

cd my-existing-project

Make sure you're in the root directory of your project.

Step 2: Initialize CodeFortify

Add CodeFortify to your existing project:

codefortify init

This analyzes your existing code and creates a configuration file.

Step 3: Analyze Current State

Get a detailed analysis of your current code quality:

codefortify score --detailed --baseline

This establishes your baseline quality score and identifies improvement areas.

Step 4: Review Suggestions

See what improvements CodeFortify recommends:

codefortify analyze --recommendations

Review the suggestions and decide which ones to apply.

Step 5: Start Enhancement

Begin the enhancement process:

codefortify enhance --incremental

CodeFortify will make incremental improvements to your existing code.

Example: Building a Todo Component

Let's create a simple todo component to see CodeFortify in action:

Interactive Code Example

Here's a basic todo component that CodeFortify will enhance:

// Before: Basic component
import React, { useState } from 'react';

function TodoApp() {
  const [todos, setTodos] = useState([]);
  const [input, setInput] = useState('');

  const addTodo = () => {
    if (input.trim()) {
      setTodos([...todos, { id: Date.now(), text: input, done: false }]);
      setInput('');
    }
  };

  return (
    <div>
      <input value={input} onChange={(e) => setInput(e.target.value)} />
      <button onClick={addTodo}>Add Todo</button>
      <ul>
        {todos.map(todo => (
          <li key={todo.id}>{todo.text}</li>
        ))}
      </ul>
    </div>
  );
}

After running codefortify enhance, this becomes:

// After: AI-enhanced component
import React, { useState, useCallback, useMemo } from 'react';
import type { Todo } from '../types/todo';

interface TodoAppProps {
  initialTodos?: Todo[];
  onTodoChange?: (todos: Todo[]) => void;
}

export const TodoApp: React.FC<TodoAppProps> = ({ 
  initialTodos = [], 
  onTodoChange 
}) => {
  const [todos, setTodos] = useState<Todo[]>(initialTodos);
  const [input, setInput] = useState<string>('');

  const addTodo = useCallback(() => {
    if (input.trim()) {
      const newTodo: Todo = {
        id: Date.now(),
        text: input.trim(),
        completed: false,
        createdAt: new Date()
      };
      
      const updatedTodos = [...todos, newTodo];
      setTodos(updatedTodos);
      setInput('');
      onTodoChange?.(updatedTodos);
    }
  }, [input, todos, onTodoChange]);

  const completedCount = useMemo(() => 
    todos.filter(todo => todo.completed).length, 
    [todos]
  );

  return (
    <div className="todo-app">
      <h1>Todo App ({completedCount}/{todos.length})</h1>
      <div className="input-group">
        <input 
          value={input} 
          onChange={(e) => setInput(e.target.value)}
          placeholder="Add a new todo..."
          className="todo-input"
        />
        <button onClick={addTodo} className="add-button">
          Add Todo
        </button>
      </div>
      <ul className="todo-list">
        {todos.map(todo => (
          <TodoItem key={todo.id} todo={todo} />
        ))}
      </ul>
    </div>
  );
};
